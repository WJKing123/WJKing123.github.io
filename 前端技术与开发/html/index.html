Welcome to nginx!

If you see this page, the nginx web server is successfully installed and working. Further configuration is required.

For example, you can add a new virtual host using the following steps:

  1. Create a new configuration file in /etc/nginx/conf.d/ named after your domain, e.g. example.com.conf.
  2. Add the following content to the file: 
    server {
	listen 80;
	server_name example.com;

	location / {
		root /var/www/example.com;
		index index.html index.htm;
	}
}
  3. Create the directory /var/www/example.com and place your website files in it.
  4. Restart nginx using the command sudo systemctl restart nginx.

You can replace example.com with your own domain name, and adjust the file paths and index file names as needed.
